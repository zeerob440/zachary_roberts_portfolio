toggle.js
toggle.js is a program that determines which "skin" is active from the browser, listens for a click from class styleToggle button in root/index.html,it assigns "modern" if nothing is set in the browser. If the current value in "skin" is "retro" it assigns "modern" and redirects to the homepage. Otherwise, it applies "retro" and redirects the user to transition/index.html. This program was chalenging to write, but my experience with Python made me aware of which structures to write. The most important concept I learned was how to store information to the browser and retrieve it. GitHub_toggle_javascript

loader.js
loader.js determines which CSS file the browser should load. By checking the saved "skin" value in localStorage (modern as default if none exists). The program also detects if it is running locally or is hosted on Github pages, it uses an if/else structure to modify the file path before loading modern.css or retro.css. GitHub_loader_javascript
