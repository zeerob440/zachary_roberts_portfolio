toggle.js

toggle.js is a program that determines which "skin" is active from the browser, listens for a click from class styleToggle button in root/index.html,it assigns "modern" if nothing is set in the browser. If the current value in "skin" is "retro" it assigns "modern" and redirects to the homepage.Otherwise, it applies "retro" and redirects the user to transition/index.html. This program was hard to write as I am just starting out in JavaScript,but from Python I knew I required either a Boolean condition test or an if-else structure to make it function, the biggest thing I learned was how to store information to the browser and retrieve it.

loader.js

loader.js determines which CSS file the browser should load. It checks the saved "skin" value in localStorage (modern as default). Next, it determines the current file depth in the folder structure by splitting the file path into a list. It subtracts the first and last elements of the list to calculate directory depth. A for loop then builds a prefix of "../" repeated for each level. Finally, it concatenates the prefix with either "modern.ccs", or "retro.ccs" thereby allowing the correct stylesheet to load, despite which page calls loader.js. loader.js is distributed across all pages except transition/index.html, as it uses the static stylesheet terminal.css.
