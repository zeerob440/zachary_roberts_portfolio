These are my first JavaScript programs. These are the JavaScript programs that power my portfolio.

toggle.js
toggle.js is a program that determines which "skin" is active from the browser, listens for a click from class styleToggle button in root/index.html,it assigns "modern" if nothing is set in the browser. If the current value in "skin" is "retro" it assigns "modern" and redirects to the homepage. Otherwise, it applies "retro" and redirects the user to transition/index.html. This program was challenging to write, but my experience with Python made me aware of which structures to write. The most important concept I learned was how to store information to the browser and retrieve it.
wht_rbt_obj.js
wht_rbt_obj.js is a for loop that prints directly to a webpage with innerHTML. Once it finishes, it redirects to root/index.html with retro.ccs applied. It is named after a program in my favorite book.
loader.js
loader.js determines which CSS file a browser should load, by checking the saved "skin" value in localStorage (modern as default if none exists). The program also detects if it is running locally or is hosted on Github pages, it uses an if/else structure to modify the file path before loading modern.css or retro.css.
